# 🎹 Simple Piano Auto Player

用 Python 實現的簡譜自動演奏工具  
透過鍵盤模擬，播放自訂格式的音樂

---

## 功能

- 支援單手 / 雙手 / 三手演奏
- 支援和弦 `[ ]`
- 支援延音 `-`
- 支援節拍細分 `.`
- 自訂簡譜格式

---

## 簡譜規則

| 符號 | 說明 |
|------|------|
| 1~7  | 音符 |
| ^ _  | 升 / 降 |
| []   | 和弦 |
| 0    | 休止（放開） |
| -    | 延續上一個音 |
| .    | 時間細分（0.5, 0.25...） |
